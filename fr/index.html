Description du Projet :

“Battle For Supremacy” est un jeu de combat en 2D, jouable à deux en multijoueur local dans lequel s’affrontent deux divinités afin de dominer le terrain.

Le jeu se déroule sur une carte évolutive, avec un concept innovant de barre de vie “commune” appellée barre de Domination. Quand un joueur met un coup, la barre de Domination augmente, quand il prend un coup, elle diminue. Le premier joueurs à atteindre les 100% de domination remporte la victoire.

Lorsque la barre de Domination d’un des joueurs augmente, la carte de jeu évolue afin de représenter (de plus en plus) le côté du joueur ayant l’avantage.

  • Réalisation :

    • Date : Mars 2016 (1ère année d’étude)
    • Durée de production : 8 semaines
    • Langage : C
    • Moteur/Librairie : SFML

  • Équipe :

    • 3 Artistes
    • 3 Programmeurs

  • Ma contribution :

    • Moteur de particule
    • Intégration d’effets de particules
